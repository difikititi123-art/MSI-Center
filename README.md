## 📌 Описание проекта
Данный репозиторий содержит подробное руководство по установке, настройке и оптимизации фирменного программного обеспечения **MSI Center**. 

**MSI Center** — это централизованное приложение от компании MSI, предназначенное для управления компонентами материнских плат, мониторингом системы, автоматического обновления драйверов (через Live Update) и тонкой настройки производительности ПК.

---

## 🚀 Основной функционал MSI Center
* **Live Update:** Автоматический поиск, скачивание и установка актуальных драйверов чипсета, сети и звука.
* **Hardware Monitor:** Отслеживание температуры процессора, частот и работы вентиляторов в реальном времени.
* **User Scenario:** Переключение режимов производительности (Extreme Performance, Balanced, Silent).
* **Mystic Light:** Управление RGB-подсветкой компонентов.
